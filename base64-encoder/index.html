What is Base64?

Base64 is a text encoding standard that represents binary data in an ASCII string format. It's widely used in data transmission, email attachments, and web APIs.

Common Uses

  • Email attachments encoding
  • Data URI schemes in HTML/CSS
  • API authentication tokens
  • Image embedding in web pages
  • Database binary data storage
