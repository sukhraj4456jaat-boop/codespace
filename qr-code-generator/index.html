Frequently Asked Questions

What is a QR code?
+
QR (Quick Response) code is a 2D barcode that stores information readable by smartphones and QR scanners.
What can I encode?
+
URLs, text, emails, phone numbers, vCards, WiFi credentials, and more. Any text up to 4,296 characters.
What's error correction?
+
If the QR code is partially damaged or obscured, error correction lets phones still scan it. Higher levels provide more protection.
Should I use PNG or SVG?
+
Use PNG for web images. Use SVG for printing to maintain quality at any size.
