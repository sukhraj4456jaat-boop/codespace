Markdown Syntax Guide

  • # Heading 1 - Create main heading
  • ## Heading 2 - Create subheading
  • **bold** - Make text bold
  • *italic* - Make text italic
  • [Link](url) - Create hyperlink
  • - Item - Create bullet list
  • \`code\` - Inline code
  • \`\`\`language code \`\`\` - Code block
